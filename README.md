## 커밋 메시지 규칙
```
1. 커밋 메시지 제목은 Gitmoji를 활용하며, 동사 원형으로 시작할 것
2. 커밋 메시지는 50자 이내로 작성할 것
3. 마지막에는 마침표를 포함한 특수 문자는 사용하지 않을 것

영어로 작성 시:
- 첫 글자는 대문자로 작성
- "Fix", "Add", "Change"의 명령어로 시작

한글로 작성 시:
- "고침", "추가", "변경"의 명령어로 시작
```
